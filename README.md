## ✨ Features

| Playback | Audio |
|---|---|
| Stream any song from YouTube Music | Audio normalization |
| Background playback | Tempo & pitch control |
| Download & cache for offline use | 10-band Equalizer with AutoEQ |
| Skip silence | Crossfade |
| Sleep timer & Alarm | Loudness control |

| Lyrics & Discovery | Library & Account |
|---|---|
| Live synced lyrics (multiple sources) | Full library management |
| AI-powered lyrics translation | Local & online playlists |
| Personalized quick picks | Import/Export playlists |
| Charts, Mood & Genres | YouTube Music account sync |
| Music recognition (ShazamKit) | Podcast support |

| Interface | Social |
|---|---|
| Light / Dark / Black / Dynamic themes | Listen Together |
| 19+ color palettes | Discord Rich Presence |
| Material 3 Design | Last.fm scrobbling |
| Home screen widgets | Share lyrics |
| Android Auto support | |
